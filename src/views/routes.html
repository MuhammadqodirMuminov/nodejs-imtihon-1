Routes

E-commerce Api comes with a set of 4 common resources:

  1. /categories GET, POST, DELETE, PUT
  2. /subcategories GET, POST, DELETE, PUT
  3. /products GET, POST, DELETE, PUT

Filters

E-commerce Api has more filters of 6 common resources:

  1. /products?categoryId=1 filter with categoryId
  2. /products?subCategoryId=2 filter with subCategoryId
  3. /products?subCategoryId=1&model=samsung filter with subCategoryId and model
  4. /products?model=samsung filter with model
